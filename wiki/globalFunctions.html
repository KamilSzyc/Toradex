GlobalFunctions

Klasa statyczna z funkcjami dostępnymi z każdego miesca w programie

Plik .*h

#include < string >
#include < sstream >

class GlobalFunctions{
	public:
		static int 			stringToInt( std::string value );
		static std::string	intToString( int value );
};

Opis:

  • stringToInt( std::string value ) - zamiana string value na int -> return
  • intToString( int value ) - zamiana int value na string -> return

Przykład:

value = GlobalFunctions::stringToInt( resultString );
