git = smartGIT + github.com

W naszym przypadku korzystamy z gita na Ubuntu za pomocą programu smartGit. Aby go używać należy zrealizować poniższe kroki:

  • Instalujemy smartGit
  • Ściągamy najnowszą wersję JAVA
  • Podmieniamy /usr/bin/java na dowiązanie symboliczne do ściągniętej javy np. /usr/java/.../bin/java
  • Teraz, aby korzystać z eclipse, należy wywołać go ze starszą wersją javy za pomocą polecenia: eclipse -vm /usr/lib/jvm/default-java/bin

Cała nasze repozytorium znajduje się na serwerach github.com. Jednak nas to, aż tak nie zajmuje, gdyż korzystamy z smartGITa, który wszystko co związane z gitem robi za nas. W celu dodania nowego repozytorium do tego programu należy podać serwer github-a, login: KamilSzyc oraz token, który można uzyskać po zalogowaniu się do serwisu. Hasło standardowe. Gdy wszystko dodamy mamy do wyboru 3 podstawowe operacje:

  • Pull - pobieramy dane z serwera( należy zawsze wykonać przed przystąpieniem do prac i przed użyciem push )
  • Push - wrzucamy dane na serwer
  • Commit - "zapisujemy wersję lokalnie, która zostanie wrzucona po zrobieniu push"
