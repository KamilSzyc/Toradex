FileLinux

Klasa do wpisywania i odczytywania wartości z pliku

Plik .*h

#include < fstream >
#include < string >
#include < iostream >

#include "../../other/GlobalFunctions.h"

class FileLinux{
	public:
		bool read( std::string path, std::string &value );
		bool read( std::string path, int &value );

		bool write( std::string path, std::string value );
		bool write( std::string path, int value );
};

Opis:

  • read( std::string path, std::string &value ) - czyta z pliku path do zmiennej value - poprzez wskaźnik - zwracana wartość bool czy operacja się powiodła
  • write( std::string path, std::string value ) - zapisuje wartość value do pliku path - zwracana wartość bool czy operacja się powiodła
  • read( std::string path,int &value ) - czyta z pliku path do zmiennej value - poprzez wskaźnik do int-a - zwracana wartość bool czy operacja się powiodła
  • write( std::string path, int value ) - zapisuje wartość int value do pliku path - zwracana wartość bool czy operacja się powiodła

Przykład:

this->fileLinux.write( "/sys/class/gpio/export", pad );
